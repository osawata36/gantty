# ガントチャートビュー

ガントチャートビューは、タスクの日程をタイムライン上にバーとして視覚的に表示するビューです。プロジェクト全体のスケジュールを俯瞰し、タスク間の時間的な関係を把握するのに最適です。

## 画面レイアウト

ガントチャートビューは左右に分かれた2ペイン構造です。

### 左ペイン：WBSリスト

| 要素 | 説明 |
|------|------|
| タスク名 | 階層構造でタスク名を表示 |
| 折りたたみアイコン | 子タスクの表示/非表示を切り替え |

左ペインはリストビューの簡易版で、タスクの階層構造を確認しながら、対応するタスクバーを右ペインで参照できます。

### 右ペイン：ガントチャート

| 要素 | 説明 |
|------|------|
| タイムスケールヘッダー | 日付/週/月を示す目盛り |
| タスクバー | 各タスクの期間を示す横棒 |
| 今日の線 | 現在の日付を示す縦線 |
| グリッド線 | 時間単位を区切る縦線 |

### 境界の調整

左右のペインの境界線をドラッグすると、各ペインの幅を調整できます。

## タイムスケールヘッダー

チャート上部のヘッダーには、選択したスケールに応じて日付情報が表示されます。

### スケール別の表示

| スケール | 上段 | 下段 |
|----------|------|------|
| 日 | 年月 | 日付（1, 2, 3...） |
| 週 | 年月 | 週番号または週の開始日 |
| 月 | 年 | 月（1月, 2月...） |

### スケールの切り替え

ツールバーのスケール切り替えボタンで、表示スケールを変更できます。

| ボタン | スケール | 用途 |
|--------|----------|------|
| 日 | 1日単位 | 短期間の詳細スケジュール確認 |
| 週 | 1週間単位 | 中期的なスケジュール確認 |
| 月 | 1ヶ月単位 | 長期的な全体俯瞰 |

::: tip 使い分け
- **日スケール**: 今週〜来週の細かい作業予定を確認するとき
- **週スケール**: 1〜3ヶ月程度のプロジェクト計画を見るとき
- **月スケール**: 半年〜年単位の長期プロジェクトを俯瞰するとき
:::

## タスクバー

各タスクは、対応する行に横棒（タスクバー）として表示されます。

### バーの構成要素

| 要素 | 意味 |
|------|------|
| バーの左端 | タスクの開始日 |
| バーの右端 | タスクの終了日 |
| バーの長さ | タスクの所要期間 |
| バーの塗りつぶし | 進捗率を示す（塗りつぶされた部分が完了分） |
| バーの色 | ステータスに応じた色分け（設定により異なる） |

### 親タスクのバー

子タスクを持つ親タスクのバーは、以下の特徴があります。

- 開始日は最も早い子タスクの開始日
- 終了日は最も遅い子タスクの終了日
- 進捗は子タスクの進捗の加重平均

親タスクのバーは、サマリーバー（ひし形の端を持つバー）として区別して表示されることがあります。

### バーの進捗表示

タスクバーの内部には、進捗率に応じた塗りつぶしが表示されます。

```
[████████░░░░░░░░░░░░] 40%
└─完了部分─┘└─残り─┘
```

## 今日の線（Today Line）

ガントチャート上には、現在の日付を示す縦線が表示されます。

- **線の色**: 赤または目立つ色
- **位置**: 今日の日付の列に垂直に引かれる

今日の線により、以下を素早く把握できます。
- 予定より遅れているタスク（今日の線より左にバーがあるが未完了）
- 今日開始または終了するタスク
- 今後の予定タスク

## 「今日に移動」ボタン

ツールバーの「今日に移動」ボタンをクリックすると、チャートの表示位置が今日の日付にスクロールします。

::: tip 便利な使い方
長期プロジェクトで過去や未来を見ていて、現在地を見失ったときに使用します。
:::

## ドラッグ操作

ガントチャートビューでは、タスクバーをドラッグして日程を変更できます。

### バーの移動（日程のシフト）

タスクバーの中央部分をドラッグすると、開始日と終了日を同時に移動できます。

1. タスクバーの中央付近にマウスを移動（カーソルが移動アイコンに変化）
2. マウスの左ボタンを押したまま左右にドラッグ
3. 希望の位置でボタンを離す
4. 開始日と終了日が新しい日付に更新される

::: info 所要日数の維持
バーを移動しても、所要日数（期間の長さ）は維持されます。
:::

### バーのリサイズ（期間の変更）

タスクバーの左端または右端をドラッグすると、期間を変更できます。

| 操作 | 結果 |
|------|------|
| 左端を左にドラッグ | 開始日が早くなる（期間が延びる） |
| 左端を右にドラッグ | 開始日が遅くなる（期間が縮む） |
| 右端を左にドラッグ | 終了日が早くなる（期間が縮む） |
| 右端を右にドラッグ | 終了日が遅くなる（期間が延びる） |

1. タスクバーの端にマウスを移動（カーソルがリサイズアイコンに変化）
2. マウスの左ボタンを押したまま左右にドラッグ
3. 希望の日付でボタンを離す

### ドラッグ中の表示

ドラッグ中は以下の情報がツールチップで表示されます。

- 新しい開始日
- 新しい終了日
- 変更後の所要日数

## 所要日数固定モード

「所要日数固定」モードを有効にすると、終了日を変更したときに開始日が自動調整されます。

### 通常モードとの違い

| 操作 | 通常モード | 所要日数固定モード |
|------|------------|-------------------|
| 右端をドラッグ | 終了日のみ変更（所要日数が変わる） | 終了日変更 + 開始日が自動調整（所要日数は維持） |

### 使用シーン

- デッドライン（締め切り）が決まっているタスク
- 他のタスクとの依存関係を維持したいとき
- 所要日数の見積もりを変えずに日程を調整したいとき

### 有効化方法

1. タスクを選択
2. 詳細パネルの「所要日数固定」チェックボックスをオン

または

1. ツールバーの「所要日数固定」ボタンをクリック
2. 選択中のタスクに所要日数固定が適用される

## チャートの操作

### スクロール

| 操作 | 方法 |
|------|------|
| 水平スクロール | マウスホイール、または下部のスクロールバー |
| 垂直スクロール | 上下のスクロールバー、または左ペインでスクロール |

### ズーム

スケール切り替え（日/週/月）でズームレベルを変更します。

より細かいズーム調整が必要な場合は、`Ctrl + マウスホイール`（Windows）/ `Cmd + マウスホイール`（Mac）で段階的にズームできます。

## タスクの選択と詳細表示

### タスクの選択

以下のいずれかの方法でタスクを選択できます。

- 左ペインのタスク名をクリック
- 右ペインのタスクバーをクリック

選択されたタスクは、バーがハイライト表示され、右側に詳細パネルが表示されます。

### 詳細の編集

タスクを選択すると、詳細パネルで以下の情報を編集できます。

- タスク名
- 開始日・終了日
- 所要日数
- 進捗率
- ステータス
- 責任者・ボール
- 備考

## 表示のカスタマイズ

### 非稼働日の表示

設定により、土日や祝日を非稼働日として背景色を変えて表示できます。

| 日 | 背景色 |
|----|--------|
| 平日 | 通常（白） |
| 土曜日 | 薄い青 |
| 日曜日・祝日 | 薄いピンク |

### 依存関係の表示

タスク間に依存関係が設定されている場合、矢印線で接続を表示できます。

- 先行タスクのバー終端から
- 後続タスクのバー始端へ
- 矢印線が引かれる

## よくある使い方

### プロジェクトスケジュールの確認

1. 月スケールで全体を俯瞰
2. 気になる期間を見つけたら週/日スケールに切り替え
3. 今日の線を基準に進捗を確認

### 日程の調整

1. 日スケールに切り替えて詳細を表示
2. 調整したいタスクバーをドラッグ
3. または、タスクを選択して詳細パネルで日付を直接入力

### マイルストーンの設定

1. 所要日数0日のタスクを作成
2. ガントチャート上で菱形アイコンとして表示される
3. 重要な日付やイベントをマークする目印として使用
