# ビューの概要

Ganttyでは、プロジェクトのタスクを3つの異なるビューで表示・編集できます。それぞれのビューは異なる目的に最適化されており、作業内容に応じて切り替えることで効率的にプロジェクトを管理できます。

## 3つのビュー

### リストビュー

タスクを階層的なリスト形式で表示します。WBS（作業分解構成図）の作成・編集に最適です。

- タスクの追加・編集・削除
- 親子関係（階層構造）の管理
- インライン編集による素早い入力
- ドラッグ&ドロップによる並び替え・階層変更

::: tip 使用シーン
- プロジェクト開始時のタスク洗い出し
- WBSの構築・整理
- タスクの詳細情報を一覧で確認したいとき
:::

[リストビューの詳細を見る](./list-view)

### ガントチャートビュー

タスクの日程をタイムライン上にバーとして表示します。スケジュール管理に最適です。

- 視覚的な日程把握
- タスクバーのドラッグによる日程変更
- 日/週/月のスケール切り替え
- 今日の日付を示すライン表示

::: tip 使用シーン
- プロジェクト全体のスケジュール確認
- マイルストーンの設定と追跡
- タスク間の時間的な関係を把握したいとき
:::

[ガントチャートビューの詳細を見る](./gantt-view)

### カンバンビュー

タスクをステータス別のカラムに分けてカード形式で表示します。進捗管理に最適です。

- ステータス別のタスク一覧
- ドラッグ&ドロップによるステータス変更
- 各ステータスのタスク数を一目で把握
- 末端タスク（子タスクを持たないタスク）のみ表示

::: tip 使用シーン
- 日々の進捗確認・更新
- チームでの作業状況共有
- ボトルネックの発見
:::

[カンバンビューの詳細を見る](./kanban-view)

## ビューの切り替え方法

画面上部のツールバーにあるビュー切り替えボタンをクリックして、表示するビューを切り替えます。

| ボタン | ビュー | アイコン |
|--------|--------|----------|
| リスト | リストビュー | リストアイコン |
| ガント | ガントチャートビュー | チャートアイコン |
| カンバン | カンバンビュー | カードアイコン |

::: info キーボードショートカット
- `Ctrl + 1` (Windows) / `Cmd + 1` (Mac): リストビューに切り替え
- `Ctrl + 2` (Windows) / `Cmd + 2` (Mac): ガントチャートビューに切り替え
- `Ctrl + 3` (Windows) / `Cmd + 3` (Mac): カンバンビューに切り替え
:::

## ビューの使い分け

各ビューは同じプロジェクトデータを異なる視点で表示しています。一方のビューで行った変更は、他のビューにも即座に反映されます。

### 作業フェーズ別の推奨ビュー

| フェーズ | 推奨ビュー | 理由 |
|----------|------------|------|
| 計画段階 | リストビュー | タスクの洗い出しと階層構造の整理に最適 |
| スケジューリング | ガントチャートビュー | 日程の設定と調整を視覚的に行える |
| 実行段階 | カンバンビュー | 日々の進捗管理とステータス更新に便利 |
| 進捗報告 | ガントチャートビュー | 全体の進捗状況を一目で把握できる |

### 役割別の推奨ビュー

| 役割 | 推奨ビュー | 理由 |
|------|------------|------|
| プロジェクトマネージャー | ガントチャート | 全体スケジュールの把握と調整 |
| チームリーダー | カンバン | チームの作業状況の把握 |
| メンバー | リスト/カンバン | 担当タスクの確認と進捗更新 |
