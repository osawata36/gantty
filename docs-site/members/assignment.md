# タスクへの割り当て

タスクにメンバーを割り当てることで、誰が責任を持ち、誰が今動くべきかを明確にできます。このページでは、責任者とボールの設定方法を説明します。

## 責任者（Responsible）とは

責任者は、**タスク完了の最終責任を負う人物**です。

### 責任者の役割

- タスクの品質を担保する
- 期限を管理し、遅延を防ぐ
- タスクの完了を承認する
- 問題が発生した際の最終判断を行う

### 責任者の特徴

| 特徴 | 説明 |
|------|------|
| 基本的に固定 | タスクのライフサイクルを通じて変更されないことが多い |
| 1タスク1責任者 | 責任を明確にするため、1つのタスクに1人の責任者 |
| 説明責任 | 進捗報告やエスカレーションの窓口となる |

::: tip ヒント
責任者は必ずしも作業者である必要はありません。プロジェクトマネージャーやリーダーが責任者となり、実際の作業はボール保持者が行うこともあります。
:::

## ボール（Ball）とは

ボールは、**現在アクションを起こすべき人物**です。スポーツのボールを持っている人が動くように、タスクを前に進めるために「今動くべき人」を示します。

### ボールの役割

- 現在のタスクの作業を遂行する
- 作業完了後、次の担当者へボールをパスする
- タスクの進捗に直接責任を持つ

### ボールの特徴

| 特徴 | 説明 |
|------|------|
| 流動的 | 作業の進行に応じて次々とパスされる |
| 1タスク1ボール | 同時に複数人がボールを持つことはない |
| 可視化 | 誰のところでタスクが止まっているか一目でわかる |

### ボールが示すもの

ボールを見ることで、以下のことがわかります。

- **停滞箇所**: どこでタスクが止まっているか
- **次のアクション**: 誰が動けばタスクが進むか
- **作業負荷**: 誰が多くのボールを抱えているか

## 責任者とボールの設定方法

タスクに責任者とボールを設定する方法を説明します。

### 詳細パネルから設定

最も一般的な設定方法です。

1. 設定したいタスクをクリックして選択します
2. 画面右側に詳細パネルが表示されます
3. 「**責任者**」のドロップダウンからメンバーを選択します
4. 「**ボール**」のドロップダウンからメンバーを選択します

変更は自動的に保存されます。

### 設定できる項目

| 項目 | 説明 | 必須 |
|------|------|------|
| 責任者 | タスクの最終責任者 | いいえ |
| ボール | 現在の担当者 | いいえ |

::: info 補足
責任者・ボールは任意項目です。小規模なプロジェクトや個人での使用時は、設定しなくても問題ありません。
:::

### 初期値について

新しくタスクを作成した場合、責任者とボールは「未設定」の状態になります。必要に応じて、タスク作成後に設定してください。

## ボールのパス

タスクの進行に応じて、ボールを次の担当者に渡す操作を「ボールのパス」と呼びます。

### パスの流れ

```
作業者A（実装）→ レビュアーB（レビュー）→ 作業者A（修正）→ マネージャーC（承認）
```

### パスの方法

1. タスクの詳細パネルを開きます
2. 「**ボール**」のドロップダウンをクリックします
3. 次の担当者を選択します

### よくあるパスのパターン

| シナリオ | パスの流れ |
|----------|-----------|
| 開発タスク | 開発者 → レビュアー → 開発者（修正）→ テスター |
| ドキュメント作成 | 作成者 → レビュアー → 作成者（修正）→ 承認者 |
| 問い合わせ対応 | 受付担当 → 対応担当 → 受付担当（報告）→ 完了 |

::: tip ヒント
ボールをパスする際は、次の担当者に作業内容を伝えることをお勧めします。タスクのメモ欄にコメントを残しておくと、引き継ぎがスムーズになります。
:::

## リストビューでの表示

リストビューでは、責任者とボールがバッジとして表示されます。

### 表示形式

- **責任者**: メンバーの名前と設定された表示色で表示
- **ボール**: メンバーの名前と設定された表示色で表示

### 列の表示/非表示

1. リストビューの列ヘッダーを右クリックします
2. 「責任者」「ボール」にチェックを入れて表示します

### 表示のカスタマイズ

列の順序はドラッグ&ドロップで変更できます。責任者やボールを優先的に確認したい場合は、タスク名の近くに配置すると便利です。

## カンバンビューでの表示

カンバンビューのタスクカードにも、責任者とボールが表示されます。

### カードでの表示

- カードの下部にメンバーのアイコンまたは名前が表示されます
- 設定された表示色で識別できます

### フィルタリング

カンバンビューでは、特定のメンバーでフィルタリングすることも可能です。

1. カンバンビューのフィルターボタンをクリックします
2. 責任者またはボールでメンバーを選択します
3. 選択したメンバーのタスクのみが表示されます

## 運用のコツ

### 責任者を決めるタイミング

- **タスク作成時**: タスクを作成したらすぐに責任者を決める
- **計画フェーズ**: スプリント計画やマイルストーン設定時にまとめて決める

### ボールの管理

- **毎日の確認**: 自分が持っているボールを毎日確認する
- **滞留の防止**: 2〜3日以上ボールを持ち続けている場合は、進捗確認やサポートを検討
- **明示的なパス**: ボールを渡す際は、次の担当者に通知する

### チームでの運用ルール例

| ルール | 説明 |
|--------|------|
| 作成者=責任者 | タスクを作成した人が責任者になる |
| 朝会でボール確認 | デイリースタンドアップで各自のボールを確認 |
| 24時間ルール | ボールを受けたら24時間以内にアクションする |

## よくある質問

### Q: 責任者とボールを同じ人にできますか？

はい、できます。一人で完結するタスクでは、責任者とボールが同じ人になることも多いです。

### Q: 責任者やボールを未設定のままにできますか？

はい、未設定のままでもタスクを作成・管理できます。ただし、チームで管理する場合は設定しておくことをお勧めします。

### Q: ボールを持っているタスクだけを表示できますか？

はい、フィルター機能を使用して、特定のメンバーがボールを持っているタスクのみを表示できます。

### Q: 責任者を途中で変更できますか？

はい、いつでも変更できます。ただし、頻繁な変更は混乱を招くため、慎重に行ってください。
