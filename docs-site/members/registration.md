# メンバー登録

プロジェクトに参加するメンバーを登録し、タスクに割り当てられるようにします。このページでは、メンバーの追加、編集、削除の方法を説明します。

## メンバーの追加

新しいメンバーをプロジェクトに追加する手順を説明します。

### 追加手順

1. メンバー管理ダイアログを開きます（「編集」>「メンバー管理」）
2. 「**メンバーを追加**」ボタンをクリックします
3. メンバー情報を入力します
4. 「**追加**」または「**保存**」ボタンをクリックして確定します

追加したメンバーは、すぐにタスクの責任者やボールとして選択できるようになります。

## 名前の設定

メンバーの名前は、タスクリストやカンバンカードで表示される識別名です。

### 入力のポイント

- **一意の名前**: 他のメンバーと区別できる名前を設定しましょう
- **わかりやすい名前**: チーム内で認識しやすい名前を使用します
- **必須項目**: 名前は必須です。空欄では登録できません

### 例

| 良い例 | 理由 |
|--------|------|
| 田中 太郎 | フルネームで明確 |
| 田中T | 同姓のメンバーがいる場合に区別可能 |
| PM 佐藤 | 役割付きで役割も明確 |

::: tip ヒント
チーム内でのニックネームや社内での呼び名を使うと、日常的なコミュニケーションとの整合性が取れます。
:::

## 表示色の設定

各メンバーに表示色を設定することで、リストビューやカンバンビューでの視認性が向上します。

### 色の用途

- **責任者バッジ**: タスクの責任者として表示される際の背景色
- **ボールバッジ**: ボール保持者として表示される際の背景色
- **カンバンカード**: タスクカードでのメンバー表示

### 色の選択

1. メンバー編集画面で「**表示色**」の項目を探します
2. カラーピッカーまたはプリセットカラーから色を選択します
3. 変更を保存します

### 色の選び方のコツ

- **コントラスト**: 背景色と文字色のコントラストを考慮します
- **区別しやすさ**: 他のメンバーと似た色を避けます
- **チームルール**: チーム内で色の使い方のルールを決めておくと統一感が出ます

::: warning 注意
あまりに薄い色や、白に近い色を選ぶと、背景と区別しにくくなることがあります。
:::

## メンバーの編集

登録済みのメンバー情報を変更する方法です。

### 編集手順

1. メンバー管理ダイアログを開きます
2. 編集したいメンバーを一覧から選択します
3. 「**編集**」ボタンをクリックします（または、メンバー名をダブルクリック）
4. 情報を変更します
5. 「**保存**」ボタンをクリックして確定します

### 編集できる項目

| 項目 | 説明 |
|------|------|
| 名前 | メンバーの表示名 |
| 表示色 | バッジやカードの表示色 |

::: info 補足
メンバー情報を変更すると、そのメンバーが割り当てられているすべてのタスクの表示に反映されます。
:::

## メンバーの削除

不要になったメンバーをプロジェクトから削除する方法です。

### 削除手順

1. メンバー管理ダイアログを開きます
2. 削除したいメンバーを一覧から選択します
3. 「**削除**」ボタンをクリックします
4. 確認ダイアログで「**削除**」を選択します

### 削除時のタスクへの影響

メンバーを削除すると、そのメンバーが割り当てられていたタスクに影響があります。

| 割り当て | 削除後の動作 |
|----------|--------------|
| 責任者として割り当て | 責任者が「未設定」になります |
| ボールとして割り当て | ボールが「未設定」になります |

::: warning 注意
削除したメンバーは復元できません。削除前に、そのメンバーが責任者やボールになっているタスクがないか確認することをお勧めします。
:::

### 削除前の確認方法

メンバーを削除する前に、以下の方法で影響範囲を確認できます。

1. リストビューを表示します
2. 責任者・ボール列を表示させます
3. 該当メンバーが割り当てられているタスクを確認します

必要に応じて、削除前に別のメンバーへ責任者やボールを再割り当てしておきましょう。

## よくある質問

### Q: メンバーを後から追加できますか？

はい、プロジェクト作成後でもいつでもメンバーを追加できます。メンバー管理ダイアログから「メンバーを追加」をクリックしてください。

### Q: メンバー数に制限はありますか？

Ganttyにはメンバー数の制限はありません。プロジェクトの規模に応じて、必要なだけメンバーを登録できます。

### Q: 同じ名前のメンバーを登録できますか？

同じ名前でも登録は可能ですが、区別が難しくなるため、できるだけ一意の名前を使用することをお勧めします。
