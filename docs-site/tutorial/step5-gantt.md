# Step 5: ガントチャートを活用する

## 目標

- ガントチャートの見方を理解する
- タスクバーをドラッグして日程を調整する
- 依存関係を設定する

## ガントチャートビューに切り替える

ツールバーの「ガント」ボタンをクリックします。

![ガントチャート](/images/05_gantt_view.png)

## 画面構成

### 左側：タスク一覧

リストビューと同様のタスク階層が表示されます。

### 右側：チャートエリア

- **タイムスケール**: 日付が表示されるヘッダー
- **タスクバー**: 各タスクの期間を表す横棒
- **今日の線**: 現在の日付を示す縦線

## タスクバーの見方

- **バーの長さ**: タスクの期間
- **バーの位置**: 開始日と終了日
- **バー内の色**: 進捗率

## スケジュール調整

### バー全体をドラッグ

タスクバーをドラッグすると、開始日と終了日が同時に移動します。所要日数は維持されます。

### バーの端をドラッグ

バーの左端または右端をドラッグすると、開始日または終了日のみを変更できます。

::: tip ドラッグ中のヒント
ドラッグ中は、変更後の日付がツールチップで表示されます。
:::

## 依存関係の設定

タスク間に「この作業が終わってから次を始める」という関係を設定できます。

### ドラッグで接続

1. タスクバーにマウスを合わせる
2. 右端に表示される接続ハンドル（オレンジ色の丸）をドラッグ
3. 後続タスクにドロップ

### 依存関係の種類

- **FS (Finish-Start)**: 前のタスクが終了してから開始（最も一般的）
- **SS (Start-Start)**: 同時に開始
- **FF (Finish-Finish)**: 同時に終了
- **SF (Start-Finish)**: 前のタスクが開始してから終了

## ズーム操作

- **ズームイン/アウト**: より広い/狭い期間を表示
- **今日に移動**: 現在の日付にスクロール

## 演習

1. 「ワイヤーフレーム作成」のバーをドラッグして、開始日を1日遅らせてみましょう
2. 「要件定義」から「ワイヤーフレーム作成」へ依存関係を設定してみましょう
