# Step 1: プロジェクトを始める

## 目標

- Ganttyを起動する
- 画面構成を理解する
- プロジェクトを保存する

## アプリを起動する

Ganttyを起動すると、自動的に新しいプロジェクトが作成されます。

![初期画面](/images/01_empty_list_view.png)

## 画面構成

### ツールバー

画面上部のツールバーには以下のボタンがあります：

- **ファイル操作**: 新規作成、開く、保存
- **ビュー切り替え**: リスト、ガント、カンバン、ネットワーク
- **メンバー管理**: プロジェクトメンバーの登録
- **ステータス管理**: ステータスのカスタマイズ

### メインエリア

リストビューでは、タスクが階層的に表示されます。現在はタスクがないので「タスクを追加」ボタンが表示されています。

### 詳細パネル

タスクをクリックすると、右側に詳細パネルが開きます。ここでタスクの詳細情報を編集できます。

## プロジェクトを保存する

作業内容を失わないよう、こまめに保存しましょう。

1. ツールバーの「保存」ボタンをクリック
2. 保存先とファイル名を指定
3. 「保存」をクリック

::: tip
ファイルは `.gantty` という拡張子で保存されます。
:::
