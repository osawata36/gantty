# Step 4: 日程を設定する

## 目標

- 開始日と終了日を設定する
- 所要日数を理解する
- 詳細パネルを使う

## シナリオ

「要件定義」タスクに日程を設定します。来週の月曜日から3日間の予定です。

## 詳細パネルを開く

1. 「要件定義」タスクをクリック
2. 右側に詳細パネルが開く

![詳細パネル](/images/04_task_detail_panel.png)

## 日程を設定する

詳細パネルで以下を設定します：

### 開始日を設定

1. 「開始日を設定」ボタンをクリック
2. カレンダーから日付を選択

### 終了日を設定

1. 「終了日を設定」ボタンをクリック
2. カレンダーから日付を選択

### 所要日数

開始日と終了日を設定すると、所要日数が自動計算されます。

::: info 日付計算のルール
- 開始日 + 所要日数 → 終了日
- 所要日数を変更すると終了日が再計算される
- 終了日を変更すると所要日数が再計算される
:::

## 詳細パネルの他の項目

### 説明

タスクの詳細な説明を記入できます。

### 予定工数・実績工数

作業にかかる時間（工数）を記録できます。

### ステータス

タスクの状態を設定します（未対応、処理中、完了など）。

### 進捗

完了度合いをパーセントで設定します。

## 演習

以下のタスクに日程を設定してみましょう：

| タスク | 開始日 | 所要日数 |
|-------|--------|---------|
| 要件定義 | 1月8日 | 3日 |
| ワイヤーフレーム作成 | 1月11日 | 5日 |
| サイトマップ作成 | 1月11日 | 2日 |

::: tip
複数のタスクを並行して進められる場合は、開始日を同じにします（例：ワイヤーフレーム作成とサイトマップ作成）。
:::
