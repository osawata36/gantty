# Step 6: カンバンで進捗管理

## 目標

- カンバンビューの使い方を理解する
- ステータスを変更する
- 進捗を更新する

## カンバンビューに切り替える

ツールバーの「カンバン」ボタンをクリックします。

![カンバン](/images/06_kanban_view.png)

## 画面構成

### カラム（列）

各ステータスごとにカラムが表示されます：

- **未対応**: まだ着手していないタスク
- **処理中**: 作業中のタスク
- **レビュー待ち**: 確認待ちのタスク
- **完了**: 完了したタスク

### タスクカード

各タスクがカードとして表示されます。カードには以下の情報が含まれます：

- タスク名
- 担当者
- 進捗率

::: info リーフタスクのみ表示
カンバンビューには、サブタスクを持たないタスク（リーフタスク）のみが表示されます。親タスクは表示されません。
:::

## ステータスを変更する

### ドラッグ&ドロップ

1. タスクカードをドラッグ
2. 別のカラムにドロップ

これだけでステータスが変更されます。

### 詳細パネルから

1. タスクカードをクリック
2. 詳細パネルでステータスを選択

## 進捗を更新する

タスクカードをクリックして詳細パネルを開き、進捗スライダーを調整します。

::: tip 完了ステータスとの連動
タスクを「完了」カラムに移動すると、進捗率が自動的に100%になります。
:::

## ステータスのカスタマイズ

プロジェクトに合わせて、ステータスを追加・編集できます。

1. ツールバーの「ステータス」ボタンをクリック
2. ステータス管理ダイアログで編集

![ステータス管理](/images/09_status_dialog.png)

## 演習

1. 「要件定義」タスクを「処理中」カラムにドラッグしてみましょう
2. 進捗を50%に設定してみましょう
3. 作業完了後、「完了」カラムに移動させてみましょう
