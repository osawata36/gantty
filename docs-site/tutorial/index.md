# チュートリアル

このチュートリアルでは、**Webサイトリニューアルプロジェクト**を例に、Ganttyの機能を段階的に学びます。

## シナリオ

あなたは、会社のWebサイトをリニューアルするプロジェクトのリーダーです。チームメンバー3名と共に、3ヶ月でプロジェクトを完了させる計画を立てます。

## 学習の流れ

| ステップ | 内容 | 習得スキル |
|---------|------|-----------|
| [Step 1](/tutorial/step1-start-project) | プロジェクトを始める | アプリの起動、基本操作 |
| [Step 2](/tutorial/step2-add-tasks) | タスクを追加する | タスクの作成、編集 |
| [Step 3](/tutorial/step3-hierarchy) | 階層構造を作る | サブタスク、WBS作成 |
| [Step 4](/tutorial/step4-schedule) | 日程を設定する | 開始日・終了日、所要日数 |
| [Step 5](/tutorial/step5-gantt) | ガントチャートを活用する | スケジュール調整、依存関係 |
| [Step 6](/tutorial/step6-kanban) | カンバンで進捗管理 | ステータス管理、進捗更新 |
| [Step 7](/tutorial/step7-members) | チームメンバー管理 | 担当者割り当て、ボール管理 |
| [Step 8](/tutorial/step8-network) | ネットワーク図 | 依存関係の可視化 |

## 所要時間

各ステップは5〜10分で完了できます。全体で約1時間のチュートリアルです。

## 始める前に

Ganttyアプリをダウンロードしてインストールしておいてください。

::: tip
チュートリアルは順番に進めることをお勧めしますが、興味のあるステップから始めることもできます。
:::
