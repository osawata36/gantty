# ステータス

タスクの状態を管理するステータスの設定です。

## デフォルトステータス

| ステータス | 説明 | 色 |
|-----------|------|-----|
| 未対応 | まだ着手していない | グレー |
| 処理中 | 作業中 | 青 |
| レビュー待ち | 確認待ち | 黄 |
| 完了 | 完了した | 緑 |

## ステータス管理ダイアログ

ツールバーの「ステータス」ボタンをクリックして開きます。

## ステータスの追加

1. 「ステータスを追加」ボタンをクリック
2. 名前を入力
3. 色を選択

## ステータスの編集

1. ステータス一覧から選択
2. 名前や色を変更

## ステータスの並び替え

ドラッグ&ドロップでステータスの順序を変更できます。

この順序はカンバンビューのカラムの並び順に反映されます。

## ステータスの削除

1. ステータスの横の「×」をクリック
2. 削除確認ダイアログで確認

::: warning 削除時の注意
- 最低1つのステータスは必要です
- 削除するステータスにタスクがある場合、別のステータスに再割り当てが必要です
:::

## 完了ステータス

「完了」タイプのステータスに設定すると：

- そのステータスに変更したタスクの進捗が自動で100%になる
- カンバンビューで右端に表示される
