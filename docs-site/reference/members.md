# メンバー

プロジェクトメンバー（リソース）の管理です。

## メンバー管理ダイアログ

ツールバーの「メンバー」ボタンをクリックして開きます。

## メンバーの登録

1. 「メンバーを追加」ボタンをクリック
2. 名前を入力
3. 必要に応じて色を選択

## メンバーの編集

1. メンバー一覧から選択
2. 名前や色を変更

## メンバーの削除

1. メンバーの横の「×」をクリック

::: warning
メンバーを削除すると、そのメンバーがアサインされているタスクから自動的に外れます。
:::

## 責任者とボール

### 責任者（Responsible）

タスクの最終責任者です。

- タスクが完了するまでの責任を持つ
- 通常は変更しない

### ボール（Ball）

現在そのタスクを担当している人です。

- 作業の進行に応じて変わる
- 他のメンバーにパスできる

### 使い分けの例

| シーン | 責任者 | ボール |
|--------|--------|--------|
| 作業開始 | 田中 | 田中 |
| レビュー依頼 | 田中 | 鈴木 |
| フィードバック対応 | 田中 | 田中 |
| 最終確認 | 田中 | 佐藤 |
| 完了 | 田中 | なし |

## タスクへの割り当て

1. タスクをクリックして詳細パネルを開く
2. 「責任者」でメンバーを選択
3. 「ボール」でメンバーを選択（オプション）

## 表示

### リストビュー

タスク名の横にバッジで表示：

- 責任者: 名前のバッジ
- ボール: 🏐マーク付きのバッジ

### カンバンビュー

カード内に担当者が表示されます。
