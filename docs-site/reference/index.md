## Ganttyの主要概念

### プロジェクトファイル

Ganttyでは、プロジェクトのすべての情報を `.gantty` 形式のファイルとして保存します。このファイルには以下の情報が含まれます：

- プロジェクト名と説明
- すべてのタスクとその階層構造
- メンバー情報
- ステータス設定

### タスクの基本属性

各タスクは以下の属性を持ちます：

| 属性 | 説明 |
|------|------|
| タスク名 | タスクを識別するための名前 |
| 開始日 | タスクの開始予定日 |
| 終了日 | タスクの終了予定日 |
| 所要日数 | タスクの完了に必要な日数 |
| 進捗率 | タスクの完了度合い（0-100%） |
| ステータス | タスクの現在の状態 |
| 責任者 | タスクの最終責任を負うメンバー |
| ボール | 現在タスクを進めている担当者 |

### ビューの種類

Ganttyは3種類のビューを提供しています：

1. **リストビュー** - タスクを階層的なリスト形式で表示
2. **ガントチャートビュー** - タスクを時系列のバー形式で表示
3. **カンバンビュー** - タスクをステータス別のカード形式で表示

各ビューの詳細については、[ビューの使い方](/views/)を参照してください。

## キーボードショートカット

Ganttyでは、効率的な操作のためにキーボードショートカットを用意しています。

| ショートカット | 動作 |
|---------------|------|
| `Enter` | タスクの編集を確定 |
| `Escape` | 編集をキャンセル |
| `Tab` | タスクをインデント（子タスク化） |
| `Shift + Tab` | タスクをアウトデント（階層を上げる） |
| `Delete` | 選択したタスクを削除 |
