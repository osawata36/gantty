# ファイル操作

プロジェクトの保存と読み込みです。

## ファイル形式

Ganttyのプロジェクトは `.gantty` という拡張子で保存されます。

内部形式はJSONで、テキストエディタで開くこともできます。

## 新規プロジェクト

アプリを起動すると、自動的に新しいプロジェクトが作成されます。

ツールバーの「新規」ボタンでも新しいプロジェクトを作成できます。

::: warning
現在のプロジェクトに未保存の変更がある場合、保存するか確認されます。
:::

## 保存

### 上書き保存

- ツールバーの「保存」ボタン
- キーボードショートカット: `Ctrl/Cmd + S`

### 名前を付けて保存

- ツールバーの「名前を付けて保存」ボタン
- 新しいファイル名・場所を指定

### 初回保存

まだ保存していないプロジェクトを保存する場合、「名前を付けて保存」ダイアログが表示されます。

## 開く

1. ツールバーの「開く」ボタンをクリック
2. `.gantty` ファイルを選択
3. 「開く」をクリック

## 未保存の変更

プロジェクトに未保存の変更がある場合、タイトルバーに `*` マークが表示されます。

::: tip
こまめに保存することをお勧めします。
:::

## トラブルシューティング

### ファイルが開けない

- ファイルが破損している可能性があります
- `.gantty` 拡張子のファイルか確認してください

### 保存できない

- 保存先フォルダへの書き込み権限を確認してください
- ディスクの空き容量を確認してください
