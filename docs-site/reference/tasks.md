# タスク

タスクはプロジェクトの作業単位です。

## タスクの属性

| 属性 | 説明 |
|------|------|
| 名前 | タスクの名称 |
| 説明 | 詳細な説明文 |
| 開始日 | 作業開始予定日 |
| 終了日 | 作業終了予定日 |
| 所要日数 | 作業にかかる日数 |
| ステータス | 未対応、処理中、完了など |
| 進捗 | 完了度合い（0-100%） |
| 責任者 | タスクの責任者 |
| ボール | 現在の担当者 |
| 予定工数 | 予定作業時間 |
| 実績工数 | 実際の作業時間 |

## タスクの作成

### 通常のタスク

1. 「タスクを追加」ボタンをクリック
2. タスク名を入力
3. Enter で確定

### サブタスク

1. 親タスクの「↳」アイコンをクリック
2. タスク名を入力
3. Enter で確定

## タスクの編集

### インライン編集

リストビューでは以下の項目をダブルクリックで直接編集できます：

| 項目 | 操作 |
|------|------|
| タスク名 | ダブルクリックで編集 |
| 所要日数 | ダブルクリックで編集（リーフタスクのみ） |

- Enter で確定、Escape でキャンセル

### 詳細パネル

1. タスクをクリック
2. 右側の詳細パネルで編集

## 階層構造

タスクは親子関係を持てます。

### 親タスクの特徴

- サブタスクを持つ
- 日程は子タスクから自動計算
- 進捗は子タスクの平均

### 階層の変更

- **ドラッグ&ドロップ**: タスクを別のタスクにドロップ
- **Tab キー**: 1段階下げる
- **Shift + Tab**: 1段階上げる

## 日程と所要日数

### 計算ルール

```
終了日 = 開始日 + 所要日数 - 1
```

### 入力パターン

| 入力 | 結果 |
|------|------|
| 開始日 + 所要日数 | 終了日を計算 |
| 開始日 + 終了日 | 所要日数を計算 |
| 終了日 + 所要日数 | 開始日を計算 |

### 親タスクの日程

親タスクの日程は子タスクから自動計算されます：

- 開始日 = 子タスクの最早開始日
- 終了日 = 子タスクの最遅終了日

### 親タスクの所要日数

親タスクの所要日数は、サブタスクの所要日数の合計から自動計算されます。
リストビューでは計算された日数が表示され、直接編集はできません。

## 進捗管理

### 進捗の設定

- 詳細パネルのスライダーで設定
- 0〜100%の範囲

### ステータスとの連動

- 「完了」ステータスに変更 → 進捗100%
- 進捗100%に設定 → ステータス「完了」

### 親タスクの進捗

子タスクの進捗の平均が自動計算されます。

## タスクの削除

::: warning
タスクを削除すると、そのサブタスクも一緒に削除されます。
:::

1. タスクにマウスを合わせる
2. ゴミ箱アイコンをクリック
