# ビュー

Ganttyには4つのビューがあります。目的に応じて使い分けてください。

## ビューの切り替え

ツールバーのビューボタンをクリックして切り替えます。

## リストビュー

タスクを階層的に管理するためのビューです。

### 特徴

- WBS（Work Breakdown Structure）形式でタスクを表示
- 階層構造のドラッグ&ドロップ
- インライン編集

### 操作

| 操作 | 方法 |
|------|------|
| タスク追加 | 「タスクを追加」ボタン |
| サブタスク追加 | タスク横の「↳」アイコン |
| 名前編集 | ダブルクリック |
| 日数編集 | 日数セルをダブルクリック |
| 階層変更 | ドラッグ&ドロップ / Tab / Shift+Tab |
| 折りたたみ | 「▼」クリック |
| 削除 | ゴミ箱アイコン |

::: tip 親タスクの日数
親タスクの日数は、サブタスクの日数合計から自動計算されます。
:::

## ガントチャートビュー

日程を視覚的に管理するためのビューです。

### 特徴

- 横軸に時間、縦軸にタスク
- タスクバーで期間を表示
- 今日の日付ライン

### 操作

| 操作 | 方法 |
|------|------|
| 日程移動 | バー全体をドラッグ |
| 期間変更 | バーの端をドラッグ |
| 依存関係追加 | 接続ハンドルをドラッグ |
| ズーム | ズームボタン |
| 今日に移動 | 「今日」ボタン |

### タスクバーの見方

- **バーの長さ**: タスクの期間
- **バー内の色**: 進捗率
- **矢印**: 依存関係

## カンバンビュー

ステータス別にタスクを管理するためのビューです。

### 特徴

- ステータスごとのカラム
- リーフタスクのみ表示
- ドラッグ&ドロップでステータス変更

### 操作

| 操作 | 方法 |
|------|------|
| ステータス変更 | カードをカラム間でドラッグ |
| 詳細表示 | カードをクリック |

::: info リーフタスク
カンバンビューでは、サブタスクを持たないタスク（リーフタスク）のみが表示されます。これは、実際の作業単位で進捗を管理するためです。
:::

## ネットワークビュー

タスクの構造と依存関係を視覚化するビューです。

### 2つの表示モード

| モード | 説明 |
|--------|------|
| WBS | 親子関係をマインドマップ形式で表示 |
| 依存 | 依存関係をネットワーク図で表示 |

### 特徴

- タスクをノードとして表示
- 依存関係を矢印で表示
- クリティカルパスの強調
- iPad/タブレット対応

### 操作（マウス）

| 操作 | 方法 |
|------|------|
| 依存関係追加 | 接続ハンドルをドラッグ |
| 依存関係削除 | エッジ選択 → Delete |
| ズーム | ズームボタン |
| 自動レイアウト | 「自動レイアウト」ボタン |

### 操作（タッチ/スタイラス）

| 操作 | 方法 |
|------|------|
| 選択 | タップ |
| 名前編集 | ダブルタップ |
| 階層変更（WBS） | 長押し + ドラッグ |
| サブタスク追加（WBS） | 選択後に「+」タップ |

### ノードの見方

- **タスク名**: ノード内
- **所要日数**: タスク名の下（依存モード）
- **色**: ステータス
