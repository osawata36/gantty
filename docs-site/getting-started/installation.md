# インストール

Ganttyをお使いのコンピュータにインストールする方法を説明します。

## 対応OS

Ganttyは以下のオペレーティングシステムに対応しています。

| OS | 対応バージョン | ファイル形式 |
|------|------|------|
| macOS | 10.15 (Catalina) 以降 | `.dmg` |
| Windows | Windows 10 以降 | `.exe` (インストーラー) |
| Linux | Ubuntu 20.04 以降、その他主要ディストリビューション | `.AppImage`, `.deb` |

## ダウンロード

最新版のGanttyは、GitHubのリリースページからダウンロードできます。

1. [Gantty リリースページ](https://github.com/osawata36/gantty/releases) にアクセスします
2. 最新リリース（Latest）の「Assets」セクションを開きます
3. お使いのOSに対応したファイルをダウンロードします

::: tip ヒント
安定版をお使いいただくことをお勧めします。プレリリース版（Pre-release）は開発中の機能を含むため、予期しない動作をする場合があります。
:::

## macOS へのインストール

### ダウンロードするファイル

- `Gantty-x.x.x.dmg` （x.x.x はバージョン番号）

### インストール手順

1. ダウンロードした `.dmg` ファイルをダブルクリックして開きます
2. 表示されたウィンドウで、Ganttyアイコンを「Applications」フォルダにドラッグ&ドロップします
3. Finder の「アプリケーション」フォルダから Gantty を起動します

### 初回起動時の注意

macOSのセキュリティ機能により、初回起動時に警告が表示される場合があります。

1. 「"Gantty" は、開発元を検証できないため開けません」というメッセージが表示された場合
2. 「システム設定」>「プライバシーとセキュリティ」を開きます
3. 「セキュリティ」セクションで「このまま開く」をクリックします
4. 再度 Gantty を起動します

## Windows へのインストール

### ダウンロードするファイル

- `Gantty-Setup-x.x.x.exe` （x.x.x はバージョン番号）

### インストール手順

1. ダウンロードした `.exe` ファイルをダブルクリックして実行します
2. インストーラーの指示に従って「次へ」をクリックします
3. インストール先を選択します（デフォルトのままで問題ありません）
4. 「インストール」をクリックしてインストールを開始します
5. インストール完了後、「完了」をクリックします

### 初回起動時の注意

Windows SmartScreen により、初回起動時に警告が表示される場合があります。

1. 「Windows によって PC が保護されました」というメッセージが表示された場合
2. 「詳細情報」をクリックします
3. 「実行」をクリックして Gantty を起動します

### 起動方法

- スタートメニューから「Gantty」を検索して起動
- デスクトップにショートカットが作成されている場合は、そちらからも起動できます

## Linux へのインストール

Linux では、AppImage 形式または deb パッケージを提供しています。

### AppImage を使用する場合

#### ダウンロードするファイル

- `Gantty-x.x.x.AppImage` （x.x.x はバージョン番号）

#### インストール手順

1. ダウンロードした `.AppImage` ファイルに実行権限を付与します

   ```bash
   chmod +x Gantty-x.x.x.AppImage
   ```

2. ファイルをダブルクリックするか、ターミナルから実行します

   ```bash
   ./Gantty-x.x.x.AppImage
   ```

::: tip ヒント
AppImage ファイルは `~/Applications` ディレクトリなどに移動して管理すると便利です。
:::

### deb パッケージを使用する場合（Debian/Ubuntu系）

#### ダウンロードするファイル

- `gantty_x.x.x_amd64.deb` （x.x.x はバージョン番号）

#### インストール手順

1. ターミナルでダウンロードしたファイルのあるディレクトリに移動します

2. 以下のコマンドでインストールします

   ```bash
   sudo dpkg -i gantty_x.x.x_amd64.deb
   ```

3. 依存関係のエラーが発生した場合は、以下のコマンドを実行します

   ```bash
   sudo apt-get install -f
   ```

4. アプリケーションメニューから Gantty を起動できます

## 初回起動

インストールが完了したら、Gantty を起動してみましょう。

1. お使いの OS の方法で Gantty を起動します
2. 起動すると、自動的に新しいプロジェクト「無題のプロジェクト」が作成されます
3. すぐにタスクの追加を開始できます

初回起動後の操作については、[最初のプロジェクト](/getting-started/first-project)で詳しく説明します。

## アップデート

新しいバージョンがリリースされた場合は、同じ手順で最新版をダウンロード・インストールしてください。既存のプロジェクトファイル（`.gantty`ファイル）は、アップデート後もそのまま使用できます。

## アンインストール

### macOS

1. Finder で「アプリケーション」フォルダを開きます
2. Gantty をゴミ箱にドラッグします

### Windows

1. 「設定」>「アプリ」>「インストールされているアプリ」を開きます
2. Gantty を見つけて「アンインストール」をクリックします

### Linux (deb)

```bash
sudo apt-get remove gantty
```

### Linux (AppImage)

AppImage ファイルを削除するだけでアンインストールできます。
