# 最初のプロジェクト

Ganttyを起動してから、タスクを追加し、プロジェクトを保存するまでの基本的な流れを説明します。

## 起動すると自動的にプロジェクトが作成される

Ganttyを起動すると、すぐに作業を開始できるよう、自動的に新しいプロジェクトが作成されます。

- プロジェクト名は「**無題のプロジェクト**」として設定されます
- 空のタスクリストが表示され、すぐにタスクを追加できます
- プロジェクトを保存するまでは、このプロジェクトはファイルとして保存されていません

::: tip ポイント
起動時の自動プロジェクト作成により、「新規作成」の操作なしですぐにアイデアを入力できます。プロジェクト名は保存時に設定できるので、まずはタスクの入力に集中しましょう。
:::

## タスクを追加する

プロジェクトが作成されたら、タスクを追加してみましょう。

### タスクの追加手順

1. 画面下部または空のリスト上にある「**タスクを追加**」ボタンをクリックします
2. タスク名を入力します（例：「企画書を作成する」）
3. **Enter** キーを押して確定します

タスクがリストに追加されます。続けてタスクを追加する場合は、同じ操作を繰り返します。

### 複数のタスクを素早く追加するコツ

- タスク追加後、続けて「タスクを追加」をクリック（または Enter キーで追加し続ける）
- まずはタスク名だけを入力し、日程などの詳細は後から設定すると効率的です

### サブタスクの追加

タスクにサブタスク（子タスク）を追加して、階層構造を作ることもできます。

1. 親となるタスクを選択します
2. 「**サブタスクを追加**」ボタンをクリックします
3. サブタスク名を入力して Enter キーを押します

サブタスクは親タスクの下にインデントされて表示されます。

## タスクの詳細を設定する

追加したタスクには、日程や担当者などの詳細情報を設定できます。

### 詳細パネルを開く

1. 設定したいタスクをクリックして選択します
2. 画面右側に詳細パネルが表示されます

### 設定できる項目

| 項目 | 説明 |
|------|------|
| タスク名 | タスクの名前を編集できます |
| 開始日 | タスクの開始予定日 |
| 終了日 | タスクの終了予定日 |
| 所要日数 | タスクにかかる日数（開始日または終了日から自動計算） |
| 進捗率 | 0%〜100% でタスクの進捗を設定 |
| ステータス | 「未着手」「進行中」「完了」などの状態 |
| 責任者 | タスクの責任者（メンバーから選択） |
| ボール | 現在タスクを担当している人（メンバーから選択） |
| メモ | タスクに関する備考やメモ |

::: info 補足
開始日・終了日を設定すると、ガントチャートビューでタスクバーとして表示されます。
:::

## プロジェクトを保存する

作業内容を失わないよう、プロジェクトをファイルに保存しましょう。

### いつ保存するべきか

以下のタイミングで保存することをお勧めします。

- タスクをある程度入力し終えたとき
- 作業を中断するとき
- 重要な変更を加えたとき

::: warning 注意
保存せずにアプリケーションを終了すると、作業内容は失われます。こまめに保存することをお勧めします。
:::

### 初回保存の手順

初めて保存する場合は、保存先とプロジェクト名（ファイル名）を指定します。

1. メニューから「**ファイル**」>「**保存**」を選択します（または `Ctrl+S` / `Cmd+S`）
2. 「名前を付けて保存」ダイアログが表示されます
3. 保存先フォルダを選択します
4. ファイル名を入力します（例：「マーケティングプロジェクト」）
5. 「**保存**」をクリックします

ファイルは `.gantty` 拡張子で保存されます。ファイル名がそのままプロジェクト名として使用されます。

### 上書き保存

一度保存したプロジェクトは、次回からは同じファイルに上書き保存されます。

- メニューから「**ファイル**」>「**保存**」を選択
- または `Ctrl+S`（Windows/Linux）/ `Cmd+S`（macOS）

### 別名で保存

既存のプロジェクトを別の名前や場所に保存したい場合は、「名前を付けて保存」を使用します。

- メニューから「**ファイル**」>「**名前を付けて保存**」を選択

## 保存したプロジェクトを開く

保存したプロジェクトファイルは、以下の方法で開くことができます。

### メニューから開く

1. メニューから「**ファイル**」>「**開く**」を選択します（または `Ctrl+O` / `Cmd+O`）
2. ファイル選択ダイアログで `.gantty` ファイルを選択します
3. 「**開く**」をクリックします

### ファイルをダブルクリック

- エクスプローラー（Windows）/ Finder（macOS）/ ファイルマネージャー（Linux）で `.gantty` ファイルをダブルクリックすると、Gantty が起動してプロジェクトが開きます

### ドラッグ&ドロップ

- `.gantty` ファイルを Gantty のウィンドウにドラッグ&ドロップしても開くことができます

## まとめ

これで Gantty の基本的な使い方を学びました。

1. **起動** - 自動的に「無題のプロジェクト」が作成される
2. **タスク追加** - 「タスクを追加」でタスクを入力
3. **詳細設定** - タスクをクリックして日程などを設定
4. **保存** - 「保存」でファイルに保存（初回はプロジェクト名を設定）
