# はじめに

Ganttyへようこそ。このセクションでは、Ganttyのインストールから最初のプロジェクト作成までをガイドします。

## Ganttyとは

Ganttyは、WBS（Work Breakdown Structure：作業分解構成図）とガントチャートを作成・編集するためのデスクトップアプリケーションです。プロジェクトのスコープ（作業範囲）とスケジュールを視覚的に管理できます。

### 主な特徴

- **直感的な操作** - タスクの追加・編集・並べ替えをドラッグ&ドロップで簡単に行えます
- **3つのビュー** - リストビュー、ガントチャートビュー、カンバンビューを切り替えて使えます
- **階層構造** - タスクを親子関係で整理し、WBSを構築できます
- **オフラインで動作** - インターネット接続なしで利用できるデスクトップアプリです
- **ファイル保存** - プロジェクトは `.gantty` ファイルとして保存・共有できます

### こんな方におすすめ

- プロジェクトの作業内容を整理したい
- タスクのスケジュールを視覚的に管理したい
- チームメンバーへのタスク割り当てを明確にしたい
- シンプルで使いやすいプロジェクト管理ツールを探している
