# タスクの作成・編集

Ganttyでタスクを作成し、編集する方法を説明します。タスクの追加から詳細な情報の設定まで、様々な編集方法を活用できます。

## タスクの作成

タスクを作成する方法はいくつかあります。状況に応じて使い分けてください。

### 「タスクを追加」ボタンから作成

最も基本的なタスク追加方法です。

1. 画面下部にある「**タスクを追加**」ボタンをクリックします
2. 新しいタスク行が追加され、タスク名の入力状態になります
3. タスク名を入力します（例：「要件定義書を作成する」）
4. **Enter** キーを押して確定します

::: tip ポイント
タスクを連続して追加する場合は、Enter キーで確定した後、続けて「タスクを追加」ボタンをクリック（またはキーボードショートカット）で効率よく入力できます。
:::

### サブタスクを追加

既存のタスクに子タスク（サブタスク）を追加できます。

1. 親となるタスクをクリックして選択します
2. 「**サブタスクを追加**」ボタンをクリックします
3. サブタスク名を入力して **Enter** キーを押します

サブタスクは親タスクの下にインデントされて表示されます。

[階層構造について詳しく見る](/tasks/hierarchy)

### キーボードショートカット

素早くタスクを追加するためのキーボードショートカットも用意されています。

| 操作 | Windows / Linux | macOS |
|------|-----------------|-------|
| 新規タスク追加 | `Ctrl + Enter` | `Cmd + Enter` |
| サブタスク追加 | `Ctrl + Shift + Enter` | `Cmd + Shift + Enter` |

## インライン編集

リストビューでは、タスク名をその場で直接編集できます。

### タスク名のインライン編集

1. 編集したいタスクのタスク名部分を**ダブルクリック**します
2. テキスト入力状態になります
3. タスク名を編集します
4. **Enter** キーを押して確定、または **Escape** キーを押してキャンセルします

::: info 補足
インライン編集中は他のタスクをクリックすると、編集内容が自動的に確定されます。
:::

### インライン編集可能な項目

リストビュー上でインライン編集できる項目は以下の通りです。

| 項目 | 編集方法 |
|------|----------|
| タスク名 | ダブルクリックでテキスト編集 |
| ステータス | クリックでドロップダウン表示 |
| 担当者 | クリックでドロップダウン表示 |
| 進捗率 | クリックで入力欄表示 |

## 詳細パネルでの編集

タスクの全ての情報を編集するには、詳細パネルを使用します。

### 詳細パネルを開く

1. 編集したいタスクをクリックして選択します
2. 画面右側に詳細パネルが表示されます

詳細パネルでは、タスクに関するすべての情報を確認・編集できます。

### 編集できる項目

詳細パネルで設定できる項目の一覧です。

#### 基本情報

| 項目 | 説明 |
|------|------|
| タスク名 | タスクの名前。作業内容がわかりやすい名前をつけます |

#### 日程

| 項目 | 説明 |
|------|------|
| 開始日 | タスクを開始する予定日 |
| 終了日 | タスクを完了する予定日 |
| 所要日数 | タスクにかかる日数（開始日・終了日から自動計算、または手動入力） |

[日程と所要日数について詳しく見る](/tasks/dates-duration)

#### ステータスと進捗

| 項目 | 説明 |
|------|------|
| ステータス | 「未着手」「進行中」「完了」などのタスクの状態 |
| 進捗率 | 0%〜100% の範囲でタスクの完了度を設定 |

[進捗管理について詳しく見る](/tasks/progress)

#### 担当者

| 項目 | 説明 |
|------|------|
| 責任者 | タスクの責任を持つメンバー |
| ボール | 現在そのタスクを担当している（ボールを持っている）メンバー |

[メンバー管理について詳しく見る](/members/)

#### その他

| 項目 | 説明 |
|------|------|
| メモ | タスクに関する補足情報やメモ |

### 変更の保存

詳細パネルでの変更は、入力欄からフォーカスが外れたタイミング、または別のタスクを選択したタイミングで自動的に保存されます。

::: warning 注意
アプリケーションを終了する前に、プロジェクトをファイルに保存してください。保存しないと変更内容が失われます。
:::

## タスクの削除

不要なタスクを削除する方法です。

### 単一のタスクを削除

1. 削除したいタスクをクリックして選択します
2. 詳細パネルの「**削除**」ボタンをクリックします
3. 確認ダイアログで「**削除**」をクリックします

または、

1. 削除したいタスクを右クリックします
2. コンテキストメニューから「**削除**」を選択します

### 複数のタスクを削除

1. 削除したいタスクを複数選択します（`Ctrl` / `Cmd` キーを押しながらクリック）
2. 右クリックしてコンテキストメニューから「**削除**」を選択します

::: danger 警告
親タスクを削除すると、その子タスク（サブタスク）もすべて削除されます。削除前に確認してください。
:::

### 削除の取り消し

タスクの削除は取り消し（Undo）機能で元に戻すことができます。

| 操作 | Windows / Linux | macOS |
|------|-----------------|-------|
| 取り消し | `Ctrl + Z` | `Cmd + Z` |

## ベストプラクティス

タスクを効率よく作成・編集するためのヒントです。

### タスク名の付け方

- **動詞で始める** - 「作成する」「確認する」「送付する」など、作業内容が明確になります
- **具体的に書く** - 「資料作成」より「企画書の初稿を作成する」の方がわかりやすい
- **適切な粒度にする** - 大きすぎるタスクはサブタスクに分解しましょう

### 効率的な入力のコツ

1. **まずタスク名だけを入力** - 全てのタスクを先に洗い出します
2. **階層構造を作る** - タスクをグループ化してサブタスクを整理します
3. **詳細を後から設定** - 日程、担当者などを一括で設定します
