# ドラッグ&ドロップ操作

Ganttyでは、ドラッグ&ドロップ操作でタスクを直感的に操作できます。リストビューでの並べ替えや階層変更、ガントチャートビューでの日程調整など、様々な場面で活用できます。

## リストビューでのドラッグ&ドロップ

リストビューでは、タスクをドラッグして並べ替えたり、階層構造を変更したりできます。

### タスクの並べ替え

同じ階層内でタスクの順序を変更できます。

**手順：**
1. 移動したいタスクの左端にあるハンドル（ドラッグアイコン）にマウスを合わせます
2. ハンドルをクリックしたままドラッグします
3. 移動先の位置でドロップします

ドラッグ中は、挿入位置を示すラインが表示されます。

::: tip ポイント
タスク行のどこでもドラッグできますが、ハンドルを使うと誤操作を防げます。
:::

### 階層の変更（子タスク化）

タスクを別のタスクの子タスクにすることができます。

**手順：**
1. 子タスクにしたいタスクをドラッグします
2. 親にしたいタスクの上にドラッグを持っていきます
3. タスクの中央付近でドロップします

ドロップ位置によって動作が変わります。

| ドロップ位置 | 動作 |
|------------|------|
| タスクの上端付近 | そのタスクの**前**に同じ階層で挿入 |
| タスクの中央付近 | そのタスクの**子タスク**として追加 |
| タスクの下端付近 | そのタスクの**後**に同じ階層で挿入 |

::: info 視覚的なフィードバック
ドラッグ中は、ドロップ先の動作を示す視覚的なフィードバックが表示されます。
- **水平線**：同じ階層に挿入
- **ボックスのハイライト**：子タスクとして追加
:::

### 複数タスクの一括移動

複数のタスクを選択して、まとめて移動できます。

**手順：**
1. `Ctrl` キー（macOSの場合は `Cmd` キー）を押しながら複数のタスクをクリックして選択します
2. 選択したタスクのいずれかをドラッグします
3. 移動先でドロップします

::: warning 注意
- 連続していないタスクも選択できます
- 親タスクを選択すると、その子タスクも一緒に移動します
- 異なる階層のタスクを選択した場合、相対的な階層関係は維持されます
:::

### キーボードショートカットとの組み合わせ

ドラッグ中にキーボードショートカットを使うことで、操作を制御できます。

| キー | 効果 |
|------|------|
| `Ctrl` / `Cmd` | コピー（ドロップ先に複製を作成） |
| `Shift` | 同じ階層に限定（子タスク化を防ぐ） |
| `Escape` | ドラッグをキャンセル |

## ガントチャートビューでのドラッグ&ドロップ

ガントチャートビューでは、タスクバーをドラッグして日程を変更できます。

### タスクバーの移動

タスクバー全体を移動して、開始日と終了日を同時に変更できます。

**手順：**
1. 移動したいタスクバーにマウスを合わせます
2. タスクバーの中央部分をクリックしたままドラッグします
3. 新しい位置でドロップします

所要日数（期間）は維持されたまま、開始日と終了日が変更されます。

**例：**
```
移動前：04/01 - 04/05（5日間）
3日後に移動
移動後：04/04 - 04/08（5日間）
```

### 期間の変更（終了日の調整）

タスクバーの右端をドラッグして、終了日と所要日数を変更できます。

**手順：**
1. タスクバーの右端にマウスを合わせます（カーソルがリサイズカーソルに変わります）
2. 右端をクリックしたままドラッグします
3. 新しい位置でドロップします

**例：**
```
変更前：04/01 - 04/05（5日間）
右端を2日延長
変更後：04/01 - 04/07（7日間）
```

### 期間の変更（開始日の調整）

タスクバーの左端をドラッグして、開始日と所要日数を変更できます。

**手順：**
1. タスクバーの左端にマウスを合わせます（カーソルがリサイズカーソルに変わります）
2. 左端をクリックしたままドラッグします
3. 新しい位置でドロップします

**例：**
```
変更前：04/05 - 04/10（6日間）
左端を2日前倒し
変更後：04/03 - 04/10（8日間）
```

### スナップ機能

ガントチャートでのドラッグ操作には、スナップ機能があります。

| スケール | スナップ単位 |
|---------|------------|
| 日表示 | 1日単位 |
| 週表示 | 1日単位 |
| 月表示 | 1日単位 |

::: tip ポイント
`Alt` キー（macOSの場合は `Option` キー）を押しながらドラッグすると、スナップを無効にして細かい調整ができます。
:::

## カンバンビューでのドラッグ&ドロップ

カンバンビューでは、タスクカードをドラッグしてステータスを変更できます。

### ステータスの変更

タスクカードを別のステータス列にドラッグすると、そのタスクのステータスが変更されます。

**手順：**
1. 移動したいタスクカードをドラッグします
2. 別のステータス列にドロップします

タスクのステータスが、ドロップした列のステータスに変更されます。

### カード内の並べ替え

同じステータス列内でカードの順序を変更できます。

**手順：**
1. 並べ替えたいタスクカードをドラッグします
2. 同じ列内の別の位置でドロップします

[カンバンビューについて詳しく見る](/views/kanban-view)

## ドラッグ&ドロップの操作ガイド

### カーソルの種類

ドラッグ操作時のカーソルの種類と意味です。

| カーソル | 意味 |
|---------|------|
| 手のひら（グラブ） | ドラッグ可能 |
| 禁止マーク | ドロップ不可 |
| 左右矢印 | 期間の変更（リサイズ） |
| 十字矢印 | 移動可能 |

### ドロップ不可の状況

以下の状況ではドロップができません。

- **自分自身への移動**：タスクを自分の子タスクにすることはできません
- **子孫への移動**：親タスクをその子孫の子タスクにすることはできません（循環参照の防止）
- **読み取り専用タスク**：ロックされているタスクは移動できません

::: warning 注意
ドロップ不可の場所にドラッグすると、カーソルが禁止マークに変わります。その状態でドロップしても、操作はキャンセルされます。
:::

## 操作の取り消し

ドラッグ&ドロップ操作は取り消し（Undo）機能で元に戻すことができます。

| 操作 | Windows / Linux | macOS |
|------|-----------------|-------|
| 取り消し | `Ctrl + Z` | `Cmd + Z` |
| やり直し | `Ctrl + Y` または `Ctrl + Shift + Z` | `Cmd + Shift + Z` |

## タッチデバイスでの操作

タブレットなどのタッチデバイスでもドラッグ&ドロップ操作が可能です。

### 基本操作

1. 移動したいタスクを**長押し**します（約0.5秒）
2. タスクが浮き上がったら、指を離さずにドラッグします
3. 移動先で指を離してドロップします

::: info 補足
タッチデバイスでは、誤操作を防ぐために長押しでドラッグモードに入ります。
:::

## ベストプラクティス

ドラッグ&ドロップを効果的に使うためのヒントです。

### リストビューでの整理

- 関連するタスクを近くに配置してグループ化しましょう
- 実行順序に合わせてタスクを並べると、進捗確認がしやすくなります
- 階層構造を活用して、大きなタスクをサブタスクに分解しましょう

### ガントチャートでのスケジュール調整

- タスクバーを移動する前に、影響範囲（依存関係のあるタスク）を確認しましょう
- 親タスクの日程は子タスクから自動計算されるため、子タスクのバーを調整します
- 大きな変更を行う場合は、事前にプロジェクトを保存しておくと安心です

### カンバンでのステータス管理

- ステータスの遷移ルール（例：「未着手」→「進行中」→「レビュー中」→「完了」）を意識しましょう
- 頻繁にステータスを変更するタスクは、ワークフローの見直しが必要かもしれません

## トラブルシューティング

### ドラッグができない

- タスクがロックされていないか確認してください
- ハンドル（ドラッグアイコン）をクリックしているか確認してください
- ブラウザの拡張機能がドラッグ操作を妨げていないか確認してください

### 意図しない場所にドロップされてしまった

- 取り消し（`Ctrl + Z` / `Cmd + Z`）で操作を元に戻せます
- ドロップ前に視覚的なフィードバック（挿入位置のライン）を確認しましょう

### タスクバーが移動できない

- 親タスクの日程は自動計算されるため、直接移動できません
- 子タスクのタスクバーを調整してください
