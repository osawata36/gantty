# タスク管理

Ganttyのタスク管理機能について説明します。タスクはプロジェクトの基本単位であり、作業内容、日程、進捗などを管理できます。

## タスクとは

Ganttyにおけるタスクは、プロジェクトで実行すべき作業の最小単位です。各タスクには以下の情報を設定できます。

| 項目 | 説明 |
|------|------|
| タスク名 | 作業内容を表す名前 |
| 開始日 | 作業を開始する予定日 |
| 終了日 | 作業を完了する予定日 |
| 所要日数 | 作業にかかる日数 |
| 進捗率 | 0〜100%で表す完了度 |
| ステータス | 未着手、進行中、完了などの状態 |
| 責任者 | タスクの責任を持つメンバー |
| ボール | 現在作業を担当しているメンバー |
| メモ | タスクに関する補足情報 |

## タスク管理の基本

タスク管理は以下の流れで行います。

### 1. タスクを作成する

プロジェクトで実行すべき作業をタスクとして登録します。「タスクを追加」ボタンからタスクを作成し、タスク名を入力します。

[タスクの作成・編集について詳しく見る](/tasks/create-edit)

### 2. 階層構造を構築する

大きなタスクをサブタスクに分解して、WBS（作業分解構成図）を作成します。親タスクと子タスクの関係を設定することで、プロジェクトの構造を明確にできます。

[階層構造について詳しく見る](/tasks/hierarchy)

### 3. 日程を設定する

各タスクの開始日と終了日（または所要日数）を設定し、スケジュールを計画します。親タスクは子タスクの日程から自動計算されます。

[日程と所要日数について詳しく見る](/tasks/dates-duration)

### 4. 進捗を管理する

作業の進み具合を進捗率（0〜100%）で記録し、プロジェクトの状況を把握します。

[進捗管理について詳しく見る](/tasks/progress)

## タスクの操作方法

Ganttyでは、複数の方法でタスクを操作できます。

### インライン編集

リストビューでタスク名をダブルクリックすると、その場で編集できます。素早い修正に便利です。

### 詳細パネル

タスクをクリックして選択すると、画面右側に詳細パネルが表示されます。日程、進捗、担当者など、すべての情報をここで設定できます。

### ドラッグ&ドロップ

タスクをドラッグして、並び順の変更や階層構造の変更ができます。直感的な操作でタスクを整理できます。

[ドラッグ&ドロップ操作について詳しく見る](/tasks/drag-drop)
