# 保存と読み込み

Ganttyでのプロジェクトファイルの保存と読み込み操作について説明します。

## 新規プロジェクト

新しいプロジェクトを作成する方法です。

### 操作手順

1. メニューから「ファイル」>「新規プロジェクト」を選択します
2. プロジェクト名を入力します
3. 「作成」ボタンをクリックします

### キーボードショートカット

| OS | ショートカット |
|----|--------------|
| Windows/Linux | `Ctrl + N` |
| macOS | `Cmd + N` |

### 新規プロジェクトの初期状態

新規プロジェクトは以下の状態で作成されます：

- タスク: 空（タスクなし）
- メンバー: 空（メンバーなし）
- ステータス: デフォルトの4種類（未対応、処理中、レビュー待ち、完了）

::: tip ヒント
アプリケーション起動時に自動的に「無題のプロジェクト」が作成されるため、すぐにタスクの入力を開始できます。
:::

## 保存

現在のプロジェクトをファイルに保存します。

### 操作手順

1. メニューから「ファイル」>「保存」を選択します
2. 初めて保存する場合は、保存先とファイル名を指定するダイアログが表示されます
3. ファイル名を入力し、保存先を選択します
4. 「保存」ボタンをクリックします

### キーボードショートカット

| OS | ショートカット |
|----|--------------|
| Windows/Linux | `Ctrl + S` |
| macOS | `Cmd + S` |

### 上書き保存

既に保存済みのプロジェクト（ファイルパスが設定されている状態）で保存を実行すると、ダイアログなしで即座に上書き保存されます。

### 保存の確認

保存が完了すると：
- ウィンドウタイトルから `*` マークが消えます
- プロジェクトは「変更なし」の状態になります

## 名前を付けて保存

現在のプロジェクトを新しいファイルとして保存します。

### 操作手順

1. メニューから「ファイル」>「名前を付けて保存」を選択します
2. 保存先とファイル名を指定するダイアログが表示されます
3. 新しいファイル名を入力します
4. 「保存」ボタンをクリックします

### キーボードショートカット

| OS | ショートカット |
|----|--------------|
| Windows/Linux | `Ctrl + Shift + S` |
| macOS | `Cmd + Shift + S` |

### 使用例

- プロジェクトのバックアップを作成する
- 既存プロジェクトをベースに新しいプロジェクトを作成する
- 異なるバージョンを別ファイルで管理する

```
例:
元ファイル: プロジェクトA.gantty
↓ 名前を付けて保存
新ファイル: プロジェクトA_v2.gantty
```

## 開く

保存済みのプロジェクトファイルを読み込みます。

### 操作手順

1. メニューから「ファイル」>「開く」を選択します
2. ファイル選択ダイアログが表示されます
3. 開きたい`.gantty`ファイルを選択します
4. 「開く」ボタンをクリックします

### キーボードショートカット

| OS | ショートカット |
|----|--------------|
| Windows/Linux | `Ctrl + O` |
| macOS | `Cmd + O` |

### 未保存の変更がある場合

現在のプロジェクトに未保存の変更がある状態で別のファイルを開こうとすると、確認ダイアログが表示されます。

選択肢：
- **保存**: 現在のプロジェクトを保存してから開く
- **保存しない**: 変更を破棄して開く
- **キャンセル**: ファイルを開く操作をキャンセル

### ファイル形式エラー

無効なファイル形式や破損したファイルを開こうとすると、エラーメッセージが表示されます。

考えられる原因：
- ファイルが.gantty形式ではない
- ファイルが破損している
- ファイルのバージョンが非対応

## 操作のまとめ

| 操作 | メニュー | Windows/Linux | macOS |
|-----|---------|---------------|-------|
| 新規プロジェクト | ファイル > 新規プロジェクト | `Ctrl + N` | `Cmd + N` |
| 保存 | ファイル > 保存 | `Ctrl + S` | `Cmd + S` |
| 名前を付けて保存 | ファイル > 名前を付けて保存 | `Ctrl + Shift + S` | `Cmd + Shift + S` |
| 開く | ファイル > 開く | `Ctrl + O` | `Cmd + O` |

## よくある質問

### 自動保存はありますか？

現在のバージョンでは自動保存機能はありません。定期的に手動で保存することをお勧めします。

### 複数のプロジェクトを同時に開けますか？

現在のバージョンでは、一度に1つのプロジェクトのみ開くことができます。複数のプロジェクトを同時に作業する場合は、アプリケーションを複数起動してください。

### 古いバージョンのファイルは開けますか？

Ganttyは後方互換性を維持しており、古いバージョンで作成されたファイルも開くことができます。
