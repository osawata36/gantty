# チュートリアル

Ganttyを使いこなすためのステップバイステップガイド集です。実際のユースケースに沿って、機能の使い方を学びましょう。

## チュートリアル一覧

### [基本的なワークフロー](./basic-workflow)

個人でプロジェクトを管理する際の基本的な流れを学びます。

- プロジェクトの作成と保存
- タスクの追加と階層構造の作成
- 日程の設定と調整
- 進捗の記録と追跡

**所要時間**: 約15分
**対象**: Ganttyを初めて使う方

---

### [チームプロジェクト管理](./team-project)

チームでプロジェクトを管理する方法を学びます。

- メンバーの登録と役割設定
- タスクへの担当者割り当て
- カンバンビューでのステータス管理
- ガントチャートでのスケジュール俯瞰

**所要時間**: 約20分
**対象**: チームでGanttyを活用したい方

---

## 学習のヒント

::: tip 実際に手を動かしながら学ぼう
チュートリアルは、実際にGanttyを操作しながら進めることをお勧めします。各ステップを自分で試すことで、より深く理解できます。
:::

::: info サンプルプロジェクトの活用
チュートリアルでは「Webサイトリニューアル」をサンプルプロジェクトとして使用します。自分のプロジェクトに置き換えて進めても構いません。
:::
