# Gantty

Easy WBS and Gantt Editor for planning the project scope and schedule.

プロジェクトのスコープとスケジュールを計画するための、シンプルで使いやすいWBS・ガントチャートエディタです。

## Features

### WBS（作業分解構造）
- 階層構造でタスクを管理
- タスクの追加・編集・削除
- ドラッグ&ドロップで並び替え
- 折りたたみ/展開表示

### ガントチャート
- 日/週/月の表示スケール切り替え
- バーのドラッグ&ドロップで日程変更
- 進捗率の視覚的表示
- 今日の日付ラインを表示

### プロジェクト管理
- 依存関係の設定（FS/SS/FF/SF）
- マイルストーン管理
- クリティカルパスの自動計算
- リソース（担当者）の負荷管理

### ファイル操作
- 独自形式（`.gantty`）での保存・読み込み
- 自動保存機能
- PDF/Excel/CSVエクスポート

## Tech Stack

- **Framework**: [Tauri v2](https://v2.tauri.app/)
- **Frontend**: React + TypeScript
- **File Format**: JSON (`.gantty`)
