# Gantty 操作マニュアルサイト設計書

## 1. 概要

### 1.1 目的
Ganttyアプリケーションのユーザーが、機能を理解し効率的に活用できるようにするための操作マニュアルサイトを構築する。

### 1.2 対象ユーザー
- プロジェクト管理者
- チームリーダー
- タスク管理を行う個人ユーザー
- WBS・ガントチャートを初めて使用するユーザー

### 1.3 サイトの特徴
- シンプルで直感的なナビゲーション
- スクリーンショット・GIFアニメーションを活用した視覚的な説明
- ユースケースベースのチュートリアル
- 検索機能による素早いアクセス

## 3. ページ別コンテンツ設計

### 3.1 トップページ (index.html)

**目的**: Ganttyの概要を伝え、ユーザーを適切なセクションへ誘導する

**コンテンツ構成**:
1. **ヒーローセクション**
   - Ganttyのキャッチコピー：「シンプルで強力なWBS・ガントチャートエディタ」
   - アプリのスクリーンショット
   - 「はじめる」ボタン → getting-started/

2. **主要機能ハイライト**（カード形式）
   - リストビュー：タスクを階層的に管理
   - ガントチャート：日程を視覚的に把握
   - カンバンビュー：ステータス別に整理
   - メンバー管理：責任者とボールの明確化

3. **クイックスタート**
   - 3ステップで始める簡単ガイド
   - Step 1: アプリを起動（自動でプロジェクト作成）
   - Step 2: タスクを追加
   - Step 3: 日程を設定

4. **ナビゲーションカード**
   - 各セクションへのリンク

## 4. デザイン方針

### 4.1 全体スタイル
- **カラースキーム**: Ganttyアプリと統一（shadcn/ui準拠）
- **フォント**: システムフォント（sans-serif）
- **レイアウト**: レスポンシブ対応（モバイル/タブレット/デスクトップ）

### 4.2 ナビゲーション
- **ヘッダー**: ロゴ + 検索バー + メインナビ
- **サイドバー**: セクション内ナビゲーション（スクロール追従）
- **パンくずリスト**: 現在位置の表示
- **フッター**: リンク集 + コピーライト

### 4.3 コンテンツ表示
- **見出し**: 明確な階層構造（h1-h4）
- **コードブロック**: 必要に応じてシンタックスハイライト
- **画像**: 影付き、角丸、キャプション付き
- **GIFアニメ**: ドラッグ&ドロップなど動的操作の説明用
- **ヒント/注意**: アイコン付きのボックス

### 4.4 インタラクション
- **ホバー効果**: リンク、ボタン
- **アンカーリンク**: 見出しへの直リンク
- **目次**: 長いページには自動生成TOC
- **前後ナビ**: ページ下部に前後ページへのリンク

## 5. 技術的実装方針

### 5.1 静的サイトジェネレーター
**選択肢**:
1. **VitePress** (推奨)
   - Vue.js製、高速
   - Markdownベース
   - 検索機能内蔵
   - Ganttyと同じViteエコシステム

2. **Docusaurus**
   - React製
   - 検索機能（Algolia連携）
   - バージョニング対応

3. **Astro**
   - 高速
   - フレームワーク非依存

### 5.2 推奨構成（VitePress使用時）

```
docs-site/
├── .vitepress/
│   └── config.ts           # サイト設定
├── public/
│   └── images/             # 画像ファイル
├── index.md                # トップページ
├── getting-started/
│   ├── index.md
│   ├── installation.md
│   └── first-project.md
├── views/
│   ├── index.md
│   ├── list-view.md
│   ├── gantt-view.md
│   └── kanban-view.md
├── tasks/
│   └── ...
└── package.json
```

### 5.3 デプロイ
- GitHub Pages（推奨）
- Vercel
- Netlify

### 5.4 画像・メディア
- スクリーンショット: PNG形式、2x解像度
- GIFアニメ: 操作説明用、最大5秒
- 図解: SVG形式（拡大対応）
